Mission Statement:

This aplication is being designed to operate as a language-decentralized chatting progam. It enables users to send and recieve messages that will be automatically translated into any supported language of preference. This program is meant to break down language barriers, and give every user the ability to chat with anyone from anywhere.
_____________________________________________________________________________________________________________
Current Version: (v1)

Allows chat between client/server in shell terminal. Currently client/server only functions on same machine for testing purposes (uses localhosts). Future updates will support external connections.
